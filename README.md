# 🧠 Soroban Credit Scoring System

## 📌 Project Description

The Soroban Credit Scoring System is a decentralized smart contract built on the Stellar Soroban platform that enables transparent and tamper-proof credit scoring for users.

Traditional credit scoring systems are centralized, opaque, and often inaccessible. This project aims to bring **trust, transparency, and accessibility** to credit evaluation using blockchain technology.

---

## ⚙️ What It Does

This smart contract allows:

- Storing credit scores for users on-chain
- Updating scores by an authorized admin
- Publicly fetching user credit scores in a transparent way

Each user is identified by their Stellar address, and their credit score is securely stored on the blockchain.

---

## ✨ Features

- 🔐 **Admin-Controlled Updates**
  - Only the contract admin can update credit scores

- 🌍 **Decentralized & Transparent**
  - Scores are stored on-chain and can be verified by anyone

- ⚡ **Fast & Low Cost**
  - Built on Stellar’s Soroban smart contract platform

- 📊 **Simple Credit Model**
  - Basic numeric score system (can be extended)

- 🔎 **Public Access**
  - Anyone can query a user’s credit score
